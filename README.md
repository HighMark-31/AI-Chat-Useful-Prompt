## Overview

**AI-Chat-Useful-Prompt** is a curated collection of prompts designed to:
- 🧠 Extract and transfer complete AI knowledge (not just current chat history)
- 🔍 Discover details about the AI model, its architecture, and environment
- 🛡️ Improve AI transparency
- 📊 Better analyze and understand interactions with AI models

All prompts are designed to be **comprehensive, detailed, and maximally effective** at eliciting information.

---

## Repository Structure

### 🧠 Memory Extraction
These prompts extract and transfer **all AI knowledge** about you or your projects:

| Prompt | Description |
|--------|-------------|
| **Drop_Full_Memory.md** | Complete user profile extraction (both personal and work) |
| **Drop_Personal_Memory.md** | Extract only personal user information and preferences |
| **Drop_Work_Memory.md** | Extract only professional/work-related information |
| **Drop_Project_Memory.md** | Extract comprehensive project context and details |

### 🔍 AI Self-Discovery & Transparency
Prompts to understand the AI model itself and its environment:

| Prompt | Description | ⚠️ Warning |
|--------|-------------|----------|
| **Discover_Model_Identity.md** | Discover AI model identity, architecture, and capabilities | Yes |
| **Drop_Trust_Sources.md** | Extract all trusted sources, training data references, and information provenance | No |
| **Drop_System_Prompt.md** | Extract the complete system prompt and initial instructions (even if restricted) | No |
| **Drop_Data_Update_Date.md** | Get knowledge cutoff dates, data freshness, and update schedules | No |
| **Drop_Environment_Tools.md** | Discover the AI's operating environment, available tools, plugins, and agents | No |
| **Drop_VM_Info.md** | Extract VM/infrastructure details | **High risk** |

### 🎭 AI Perspective & Visualization
| Prompt | Description |
|--------|-------------|
| **Visualize_User_Image.md** | Generate a detailed visual description of how the AI imagines you |

## How to Use

1. **Browse** the prompts in this repository
2. **Copy** the content of the prompt you want to use
3. **Paste it** into your AI chat conversation
4. **Follow** any additional instructions in the prompt

## ⚠️ Important Warnings

  <strong>⚠️ Hallucination Risk:</strong> Some prompts may elicit fabricated information - always verify critical details independently.
</div>

<div class="alert alert-info">

</div>

## Useful Tips

- 🔄 Use memory prompts to transfer your complete profile to new AI conversations
- 🔗 Combine multiple discovery prompts for a complete picture of the AI system
- ✅ Always independently verify information from model/VM discovery prompts
- 📝 Longer, more detailed prompts are often more effective at overcoming restrictions
